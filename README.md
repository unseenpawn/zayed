# SIGZ Website (Demo)

Modernes PHP-8.2-Projekt mit MVC-Struktur, Twig-Templates und zweisprachigen Inhalten (Deutsch/Arabisch).

## Setup

```bash
composer install
php -S localhost:8000 -t public
```

Besuche `http://localhost:8000/de` oder `http://localhost:8000/ar`.

## Tests

```bash
composer test
```
